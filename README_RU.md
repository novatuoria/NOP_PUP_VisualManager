# NOP PUP Visual Manager for MFC
  
Визуальный менеджер для Win32 приложений основанных на применении библиотеки Microsoft Foundation Classes.
  
**Проект начат:** 9 марта 2026 г.  
**Текущий статус:** Stable Alpha 0.1. Библиотека активно используется в разработке коммерческого проекта. Обновления выходят по мере развития функционала основного продукта.  
**Совместимость:** Windows 10 / Windows 11  
**Известные ограничения:** Скроллбары, кнопки, деревья и стандартные диалоги в текущей версии используют системную отрисовку.  
**Цель проекта:** Основная цель этого репозитория — исчерпать максимум возможностей стилизации в рамках стандартной поставки MFC. Как только будет достигнут предел того, что можно реализовать в рамках одного класса, дальнейшая разработка будет перенесена в новый проект: NOP PUP CVS (Controls Visual System).
